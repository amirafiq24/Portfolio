InFlow Task Manager

InFlow AI is a web based productivity tool our team built in 24 hours during a hackathon. It uses OpenAI’s API to generate a day by day action plan from a user’s task and deadline. We developed the frontend and backend using HTML, CSS, JavaScript, and Flask in Python, and deployed the app using Vercel and Render. The app was designed to be interactive and easy to use, allowing users to adjust and regenerate plans quickly. Despite limited experience with AI integration, we delivered a fully functional project within the hackathon timeframe.
