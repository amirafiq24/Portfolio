Autonomous Path Finder Robot

Our team designed and built a line-following robot powered by Arduino that competed in the Pathfinder Competition. I was responsible for programming the robot's core behavior, enabling it to follow a black line using a color sensor while detecting obstacles via ultrasonic sensing. We implemented logic for path correction, dynamic turning, and real-time decision-making to ensure smooth autonomous navigation. The robot executed multi-angle scanning routines to recover from off-track situations and intelligently handled blocked paths. Our well-coordinated teamwork and innovative approach led us to win first place in the competition, along with a $300 prize for our performance.

Overview

  • Designed and built an autonomous line-following robot for a competition.
  • Worked with a team of four to integrate hardware, sensors, and software logic.
  • Tuned the robot to navigate paths efficiently using a TCS3200 color sensor.

Implementation

  • Programmed the Arduino to detect and follow black lines using color sensor.
  • Implemented logic for direction scanning and obstacle avoidance with ultrasonic sensors.
  • Used motor control through the L298N driver board for precise movement and turning.

Results

  • Successfully completed the course and won first place in the Pathfinder competition.
  • Earned a prize of $300 by outperforming other competing teams.
  • Robot was able to adapt and recover from path loss during live runs.

What I Learned

  • Hands-on experience with Arduino programming and real-time robot control.
  • Collaborating effectively under time pressure and solving hardware-software bugs.
  • Improved understanding of sensor integration, motor drivers, and control logic.

Technologies Used:

Arduino (C++), Arduino Uno R3, TCS3200 Color Sensor, Ultrasonic sensor, L298N Motor Driver
