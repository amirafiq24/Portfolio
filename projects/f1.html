F1 Management System

This project is a full stack Formula One data management system designed using a traditional server client relational database. It models real world F1 entities such as teams, drivers, cars, races, circuits, sponsors, and lap statistics with proper primary and foreign key relationships. Data is collected from public motorsport sources and APIs, cleaned, and populated into a MySQL database with strict referential integrity. A Python backend handles database queries and business logic, while a Streamlit frontend allows users to explore teams, sponsors, races, and driver performance interactively. The project emphasizes database design, normalization, constraints, and practical client server integration.
