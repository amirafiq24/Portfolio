ASL Hand Gesture Recognition

This project explores real time American Sign Language (ASL) recognition using a webcam and basic AI techniques. The system recognizes single hand ASL gestures by observing hand movements and patterns in live video. The goal of this project is to build an interactive and accessible gesture recognition tool while learning how AI models are trained and used in real time applications. Along the way, I gained practical experience working with data, experimenting with different ideas, and improving the system through testing.

Overview

  • Built a real time ASL hand gesture recognition system
  • Recognized basic single hand ASL gestures from live video
  • Processed hand movement data for gesture classification

Implementation

  • Used MediaPipe to track and detect hand positions from webcam input
  • Trained a deep learning model using PyTorch
  • Integrated OpenCV for real time webcam processing

Results

  • Successfully recognized single hand ASL gestures in real time
  • Produced stable and consistent gesture predictions during live use
  • Demonstrated reliable hand tracking and gesture recognition

What I Learned

  • Applying AI to real world computer vision problems
  • Working with live video data in an interactive setting
  • Integrating trained models into real time applications

Built With:

Python, PyTorch, OpenCV, MediaPipe, Computer Vision, Deep Learning
