University Social Web

This project is a university-based social platform designed to connect students through shared academic interests. Each university hosts multiple communities based on majors, where students can join, post questions, share resources, and interact with peers. Users can easily sign up, explore various communities, and create or view posts related to classes, professors, or general topics. The app features a clean, responsive interface and supports token-based authentication for secure access. It’s built with a Flask backend, React frontend, and PostgreSQL database. This project promotes academic collaboration and fosters a sense of community among students from different universities and majors.

Overview

  • Connected students through academic communities

Implementation
