University Social Web

This project is a university based social platform created to help students connect through shared academic interests. Each university includes multiple communities based on majors, where students can join discussions, ask questions, share resources, and interact with classmates. Users can explore different communities, and create or browse posts related to courses, professors, or general topics. The app has a clean and responsive design and includes secure login features to protect user access. Built with a Flask backend, React frontend, and PostgreSQL database, the platform encourages academic collaboration and helps students feel more connected across universities and majors.

Overview

  • Built social platform for university students
  • Connecting students through academic communities
  • Designed and developed full stack system from scratch

Implementation

  • Developed React frontend with reusable components
  • Populated database with universities and majors
